# Web Development Learning Journey

A 6-week intensive program to learn modern full-stack web development, culminating in a collaborative worldbuilding platform.

## Project Goal

Build a production-ready collaborative worldbuilding platform that allows multiple users to:
- Create and manage imaginary universes
- Develop characters with complex relationships
- Write collaborative stories
- Generate AI-powered content (stories, illustrations)
- Experience stories with text-to-speech and synchronized visuals

## Technology Stack

**Frontend:**
- React 18+ with TypeScript
- Tailwind CSS
- Vite

**Backend:**
- Rust with Axum framework
- PostgreSQL with sqlx
- WebSockets for real-time collaboration

**Deployment:**
- Vercel (frontend)
- Railway/Fly.io (backend)
- Supabase (database & storage)

## Timeline

- **Week 1:** Web Fundamentals (HTML, CSS, JavaScript, TypeScript, React basics)
- **Week 2:** React Deep Dive (hooks, routing, state management, forms)
- **Week 3:** Rust Backend (Axum, PostgreSQL, REST APIs)
- **Week 4:** Real-Time Collaboration (WebSockets, collaborative editing)
- **Week 5:** Capstone Part 1 (Auth, complex relationships, core features)
- **Week 6:** Capstone Part 2 (AI integration, TTS, deployment)
